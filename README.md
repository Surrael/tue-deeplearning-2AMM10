# Deep Learning - 2AMM10
Tutorials and practicals for the TU/e Deep Learning course (2AMM10)

Tutorials are complete notebooks that you can explore on your own. 

Practicals are the notebooks that will be discussed during the practical sessions on Fridays. You should try to complete them beforehand, the practical sessions then serve as Q&A sessions with the teaching assistants. After each session, an example solution will be added here as well.
